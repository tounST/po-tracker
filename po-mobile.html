วิธี Deploy Apps Script:
1. เปิด Google Sheet → Extensions → Apps Script
2. วาง code จากไฟล์ po-tracker-gas.js
3. Deploy → New deployment → Web app
4. Execute as: Me | Access: Anyone
5. Copy URL มาวางตรงนี้
